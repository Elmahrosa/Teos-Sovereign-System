# Overview Dashboard

## Purpose
Provide a high-level institutional view of TEOS Sovereign Stack pilots and modules.

---

## Governance
- Dashboard overseen by Ministry/Regulator
- Independent audit access
- MDB/DFI monitoring rights

---

## Data Sources
- Treasury transactions
- Governance proposals
- SDG-linked indicators
- FDI inflows and outflows

---

## Indicator Structure
- Each metric tagged to relevant SDG
- Event taxonomy for classification
- Real-time updates via API

---

## Access & Oversight Roles
- **Government** → Policy and compliance monitoring
- **MDBs/DFIs** → Funding and impact oversight
- **Community** → Transparency and participation
- **Auditors** → Independent verification

---

## Outcome
This dashboard ensures institutional reviewers can quickly assess **governance, compliance, and impact** across all TEOS modules.
