# FDI Framework

TEOS includes modules designed to improve transparency, monitoring, and compliance for investment flows.

## Objectives
- project-level visibility and reporting
- ESG and SDG tagging
- oversight-ready dashboards for institutions and partners

## Core Components
- T-FDI (monitoring and compliance dashboard)
- T-Audit (evidence and oversight)
- T-Infra / T-Green / T-Work (sector execution modules)
