# Governance Lifecycle

1. Proposal submission (authorized actor)
2. Eligibility validation (T-ID)
3. Compliance gate (T-Audit controls)
4. Voting and approval (T-Gov)
5. Execution logging (T-Finance / T-RWA)
6. Monitoring and reporting (Dashboards via T-API)
7. Immutable audit trail retained throughout (T-Audit)
